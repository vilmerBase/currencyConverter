Конвертер курса валют.

Простой конвертер курса валют с выборкой из 4-х национальных валют. 
Курс актуален, данные вытягиваются из открытой базы данных с регулярной частотой обновления. 
В поле ввода суммы установлен шаблонный паттерн проверки на числовой тип данных и проверкой на пустоту.

Демонстрация: http://vilmerbase.github.io/currencyConverter/
